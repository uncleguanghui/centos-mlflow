# 快速部署Mlflow(1.0.0)

本项目用于以`root`身份在Centos快速部署Mlflow(1.0.0)

# 上手指南

## clone

```
git clone https://github.com/uncleguanghui/centos-mlflow.git
cd centos-mlflow
```

进来后可以看到

```
.
├── README.md
├── deploy_centos.sh  # 部署脚本
└── mlflow.ini  # supervisor配置

0 directories, 3 files
```

## 部署

```
sh deploy_centos.sh
```
